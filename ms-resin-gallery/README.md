# ✨ Ms Resin Gallery — Product Catalog App

A beautiful product catalog web app for Ms Resin Gallery, built with React + Vite.

## Features
- 🌸 All 10 catalog products pre-loaded
- 🔍 Search products by name or description
- 🗂️ Filter by category (Frames, Gifts, Platters, Accessories)
- ➕ Add new products with image upload, name, category, price, description & badge
- 🗑️ Remove any product
- 📱 Fully responsive (mobile-friendly)
- 💛 Enquire button links directly to WhatsApp

## Setup & Run

### 1. Install dependencies
```bash
npm install
```

### 2. Start development server
```bash
npm run dev
```

Then open: **http://localhost:5173**

### 3. Build for production
```bash
npm run build
```
